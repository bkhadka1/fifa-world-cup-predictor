# FIFA World Cup 2026 Prediction Model

A machine learning pipeline that predicts match outcomes and simulates the 2026 FIFA World Cup tournament using a 4-model ensemble and 100,000 Monte Carlo simulations.

**Live dashboard:** [wc2026-predictor.streamlit.app](https://fifa-world-cup-predictor-prnvwr2hnhicdyotplcvpq.streamlit.app)

## How it works

### 1. Data
- 35+ years of international match results (1990–2026)
- Real-world ELO ratings seeded from [eloratings.net](https://www.eloratings.net) as of June 11, 2026
- Features engineered from match history: ELO ratings, recent form, head-to-head records, venue type, tournament importance

### 2. Models
| Model | Role |
|---|---|
| Poisson Regression | Predicts expected goals → win/draw/loss probabilities |
| XGBoost | 3-class outcome classifier (tuned with RandomizedSearchCV) |
| Random Forest | Ensemble member (tuned with RandomizedSearchCV) |
| Logistic Regression | Baseline classifier |
| **Weighted Ensemble** | Final predictions — weighted average of all 4 |

### 3. Tournament simulation
- Official FIFA 2026 format: 12 groups → top 2 + best 8 third-place = **32 teams**
- Round of 32 → Round of 16 → Quarter-finals → Semi-finals → Final
- **100,000 Monte Carlo simulations** — each runs the full tournament probabilistically
- All Poisson lambdas pre-computed before the simulation loop for performance (~60 seconds total)

## Setup

```bash
git clone https://github.com/bkhadka1/fifa-world-cup-predictor.git
cd fifa-world-cup-predictor

python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### Data
Download from Kaggle and place in `data/raw/`:
- [International Football Results](https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017) — `results.csv`, `shootouts.csv`
- [FIFA Rankings](https://www.kaggle.com/datasets/cashncarry/fifaworldranking) — `fifa_ranking.csv`

### Run notebooks in order

```bash
jupyter notebook
```

```
01_eda.ipynb              → explore and clean the data
02_features.ipynb         → build ELO ratings and feature matrix
03_models.ipynb           → train all 4 models
04_simulation.ipynb       → run 100,000 tournament simulations
05_tuning.ipynb           → tune XGBoost and Random Forest
06_results_comparison.ipynb  → compare predictions vs reality (after July 19)
```

### Run the dashboard locally

```bash
streamlit run dashboard/app.py
```
