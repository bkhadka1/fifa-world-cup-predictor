# FIFA World Cup Prediction Model

A machine learning pipeline that predicts match outcomes and simulates World Cup tournament results.

## Approach

- **Goals model** — Poisson regression predicts expected goals for each team
- **Match model** — XGBoost classifies win/draw/loss given team features
- **ELO ratings** — custom ELO system built from 30+ years of match history
- **Monte Carlo simulation** — 100,000 tournament simulations to estimate championship probabilities

## Project structure

```
fifa-world-cup-predictor/
├── data/
│   ├── raw/          ← place downloaded CSVs here (gitignored)
│   ├── processed/    ← cleaned data generated by notebooks
│   └── external/     ← optional extra sources
├── notebooks/
│   ├── 01_eda.ipynb          ← data loading & exploratory analysis
│   ├── 02_features.ipynb     ← ELO ratings & feature engineering
│   ├── 03_models.ipynb       ← model training & evaluation
│   └── 04_simulation.ipynb   ← tournament Monte Carlo simulation
├── src/
│   ├── utils.py      ← shared helpers & data loaders
│   ├── elo.py        ← ELO rating system
│   ├── features.py   ← feature engineering pipeline
│   ├── model.py      ← training & evaluation code
│   └── simulate.py   ← tournament bracket simulation
├── models/           ← saved model artifacts
├── outputs/          ← charts and prediction CSVs
└── requirements.txt
```

## Setup

```bash
git clone https://github.com/YOUR_USERNAME/fifa-world-cup-predictor.git
cd fifa-world-cup-predictor

python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Data

Download from Kaggle:
- [International Football Results 1872–present](https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017)

Place the three CSV files (`results.csv`, `shootouts.csv`, `goalscorers.csv`) into `data/raw/`.

## Run the notebooks in order

```
notebooks/01_eda.ipynb       → understand the data
notebooks/02_features.ipynb  → build ELO + features
notebooks/03_models.ipynb    → train models
notebooks/04_simulation.ipynb → simulate & predict
```
